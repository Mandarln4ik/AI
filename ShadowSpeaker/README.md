# ShadowSpeaker - Локальный ИИ-ассистент для помощи в диалогах

## Описание
ShadowSpeaker - это локальное приложение, которое слушает ваши диалоги (Discord, Teamspeak и др.), распознает речь, анализирует контекст и предлагает варианты ответов через прозрачный оверлей поверх экрана.

## Особенности
- 🎤 **Распознавание речи** с поддержкой пользовательских моделей Whisper (.pt)
- 🔊 **Диаризация спикеров** - определение кто говорит
- 🧠 **Память диалога** - помнит последние 5 минут беседы
- 🖥 **Захват экрана** - визуальный контекст для ситуаций в играх
- 💡 **3 варианта ответа** от локальной LLM
- ⚙ **Поддержка 3 провайдеров**: Ollama, LM Studio, llama.cpp
- 🎨 **Прозрачный оверлей** с горячими клавишами
- 🔒 **Полная приватность** - всё работает локально

## Системные требования
- **ОС**: Windows 11
- **Python**: 3.12+
- **GPU**: NVIDIA с CUDA 12.4+ (у вас CMP 50HX + GTX 1630)
- **RAM**: 16GB+ рекомендуется
- **Место**: 10GB+ для моделей

## Установка

### Шаг 1: Подготовка окружения

```bash
# Создание виртуального окружения (рекомендуется)
python -m venv venv
venv\Scripts\activate

# Или установка глобально
```

### Шаг 2: Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 3: Установка VB-CABLE (для захвата звука из приложений)

1. Скачайте VB-CABLE: https://vb-audio.com/Cable/
2. Установите и перезагрузитесь
3. В настройках звука Windows:
   - Установите VB-Cable как устройство записи по умолчанию
   - В Discord/Teamspeak настройте вывод звука на VB-Cable

### Шаг 4: Установка Ollama или LM Studio

#### Вариант A: Ollama (рекомендуется)
```bash
# Скачайте с https://ollama.ai
# Установите модель
ollama pull phi3
# Или другую модель
ollama pull llama3
ollama pull mistral
```

#### Вариант B: LM Studio
```bash
# Скачайте с https://lmstudio.ai
# Загрузите любую GGUF модель
# Запустите локальный сервер на порту 1234
```

### Шаг 5: Импорт вашей модели Whisper

Если у вас есть файл `large-v3-turbo.pt`:
1. Запустите приложение: `python main.py`
2. Нажмите Ctrl+S для настроек
3. Перейдите на вкладку "Whisper STT"
4. Нажмите "📁 Импорт .pt модели"
5. Выберите ваш файл

## Запуск

```bash
python main.py
```

### Горячие клавиши
- **Ctrl+S** - Открыть настройки
- **Ctrl+1/2/3** - Выбрать вариант ответа
- **Перетаскивание** - Перемещение оверлея

## Настройка

### Вкладка Whisper STT
- **Модель**: Выберите из списка или импортируйте .pt файл
- **Устройство**: cuda (рекомендуется) или cpu
- **Язык**: ru для русского, en для английского

### Вкладка LLM Провайдер
- **Провайдер**: ollama / lmstudio / llama_cpp
- **Модель**: Выберите из доступных
- **Base URL**: http://localhost:11434 для Ollama
- **Порт LM Studio**: 1234 по умолчанию
- **Тест подключения**: Проверьте работу

### Вкладка Аудио
- **Устройство ввода**: Выберите VB-Cable или другое
- **Частота**: 16000 Гц оптимально
- **Длительность чанка**: 5 секунд

### Вкладка Экран
- **Захват экрана**: Включить/выключить
- **Интервал**: 10 секунд между скриншотами
- **Размер**: 800x600 для оптимизации

### Вкладка Оверлей
- **Позиция**: X, Y координаты
- **Размер**: Ширина и высота окна
- **Прозрачность**: 0.8 рекомендуется

## Как это работает

1. **Захват аудио**: Приложение слушает Virtual Audio Cable
2. **Распознавание**: Whisper транскрибирует речь в текст
3. **Диаризация**: Определение спикеров (кто говорит)
4. **Память**: Сохранение последних 5 минут диалога
5. **Захват экрана**: Скриншоты каждые 10 секунд
6. **Генерация**: LLM анализирует контекст и предлагает 3 варианта
7. **Оверлей**: Показ вариантов поверх других окон

## Оптимизация для вашей системы

У вас две видеокарты:
- **NVIDIA CMP 50HX** (майнинговая, без видеовыходов)
- **NVIDIA GeForce GTX 1630** (основная)


## Troubleshooting

### Ошибка "Модель не найдена"
- Проверьте путь к файлу .pt
- Убедитесь что файл в директории data/models/

### Ошибка подключения к Ollama
- Запустите `ollama serve`
- Проверьте `http://localhost:11434/api/tags`

### Нет звука
- Проверьте настройки VB-Cable
- Убедитесь что Discord выводит звук на VB-Cable

### Оверлей не показывается
- Проверьте что окно не за пределами экрана
- Измените позицию в настройках
