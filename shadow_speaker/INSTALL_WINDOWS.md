# Инструкция по установке ShadowSpeaker на Windows 11

## Ваша конфигурация
- **ОС**: Windows 11
- **Python**: 3.13.7 ✅
- **PyTorch**: 2.6.0+cu124 ✅
- **GPU**: NVIDIA CMP 50HX + GTX 1630 (CUDA 12.4) ✅

## Шаг 1: Подготовка проекта

```powershell
# Перейдите в директорию проекта
cd C:\путь\к\ShadowSpeaker

# Создайте виртуальное окружение (рекомендуется)
python -m venv venv

# Активируйте его
.\venv\Scripts\activate
```

## Шаг 2: Установка зависимостей

```powershell
# Обновите pip
python -m pip install --upgrade pip

# Установите зависимости
pip install -r requirements.txt
```

### Если возникли ошибки с установкой:

#### Для sounddevice:
```powershell
pip install sounddevice --only-binary :all:
```

#### Для faster-whisper:
```powershell
pip install faster-whisper --only-binary :all:
```

#### Для PyQt6:
```powershell
pip install PyQt6 --only-binary :all:
```

## Шаг 3: Установка VB-CABLE (для захвата звука)

1. Скачайте VB-CABLE Driver: https://vb-audio.com/Cable/
2. Распакуйте архив
3. Запустите `VBCABLE_Setup_x64.exe` от имени администратора
4. Перезагрузите компьютер

### Настройка звука после перезагрузки:

1. Откройте **Параметры** → **Система** → **Звук**
2. В разделе **Ввод** выберите **CABLE Input (VB-Audio Virtual Cable)**
3. Откройте **Discord** → **Настройки** → **Голос и видео**
4. В **Устройство вывода** выберите **CABLE Output (VB-Audio Virtual Cable)**

Теперь звук из Discord будет идти в виртуальный кабель, который слушает ShadowSpeaker.

## Шаг 4: Установка Ollama (или LM Studio)

### Вариант A: Ollama (рекомендуется)

1. Скачайте установщик: https://ollama.ai/download/OllamaSetup.exe
2. Запустите установщик
3. После установки откройте PowerShell и выполните:

```powershell
# Проверка работы
ollama --version

# Установка модели Phi-3 (легкая и быстрая)
ollama pull phi3

# Или Llama 3 (более умная, но тяжелее)
ollama pull llama3

# Или Mistral (хороший баланс)
ollama pull mistral
```

4. Ollama автоматически запускается как сервис. Проверьте:
```powershell
curl http://localhost:11434/api/tags
```

### Вариант B: LM Studio

1. Скачайте: https://lmstudio.ai/
2. Установите и запустите
3. В поиске моделей найдите нужную (например, "Phi-3")
4. Скачайте модель в формате GGUF
5. Перейдите во вкладку **Local Server**
6. Нажмите **Start Server** (порт 1234 по умолчанию)

## Шаг 5: Импорт вашей модели Whisper

У вас есть файл `large-v3-turbo.pt`. Есть два способа его использовать:

### Способ A: Через интерфейс (рекомендуется)

1. Запустите приложение (см. Шаг 6)
2. Нажмите **Ctrl+S** для открытия настроек
3. Перейдите на вкладку **🎤 Whisper STT**
4. Нажмите **📁 Импорт .pt модели**
5. Выберите ваш файл `large-v3-turbo.pt`
6. Модель скопируется в `data/models/` и появится в списке

### Способ B: Вручную

1. Создайте директорию:
```powershell
mkdir data\models
```

2. Скопируйте ваш файл:
```powershell
copy C:\путь\к\large-v3-turbo.pt data\models\
```

3. При запуске модель автоматически обнаружится

## Шаг 7: Проверка работы

### Тест 1: Проверка оверлея
1. Нажмите **Ctrl+S** - должно открыться окно настроек
2. Измените любую настройку и нажмите **💾 Сохранить**
3. Окно должно закрыться с сообщением об успехе

### Тест 2: Проверка распознавания речи
1. Откройте настройки (**Ctrl+S**)
2. На вкладке **🔊 Аудио** выберите ваше устройство (VB-Cable)
3. Сохраните настройки
4. Скажите что-нибудь в Discord
5. В консоли должны появиться сообщения:
   ```
   [14:30:45] Спикер 1: привет как дела
   ```

### Тест 3: Проверка генерации ответов
1. После нескольких реплик в диалоге
2. Должен появиться оверлей с 3 вариантами ответа
3. Нажмите **Ctrl+1**, **Ctrl+2** или **Ctrl+3** для выбора
4. В консоли появится:
   ```
   💬 Выбран вариант 1: Привет! Всё отлично, спасибо!
   ```

## Решение проблем

### Ошибка: "No module named 'X'"
```powershell
pip install X --only-binary :all:
```

### Ошибка: "CUDA out of memory"
У вас две карты. Попробуйте указать конкретную:

В файле `config.py` добавьте в начало:
```python
import os
os.environ["CUDA_VISIBLE_DEVICES"] = "1"  # GTX 1630
```

### Ошибка: "Ollama connection failed"
```powershell
# Проверьте что Ollama запущена
ollama serve

# Проверьте доступность
curl http://localhost:11434/api/tags
```

### Оверлей не появляется
1. Нажмите **Ctrl+S**
2. На вкладке **📟 Оверлей** измените позицию X=100, Y=100
3. Сохраните и перезапустите приложение

### Нет звука из Discord
1. Проверьте что VB-Cable установлен
2. В Discord: Настройки → Голос и видео → Устройство вывода → CABLE Output
3. В Windows: Параметры → Система → Звук → Ввод → CABLE Input

### Медленная работа
1. Используйте более легкую модель LLM (Phi-3 вместо Llama 3)
2. Уменьшите частоту захвата экрана до 20 секунд
3. Отключите захват экрана если не нужен

## Оптимизация для вашей системы

### Использование GTX 1630 вместо CMP 50HX

CMP 50HX - майнинговая карта без видеовыходов, может быть не оптимальна для инференса.

Для принудительного использования GTX 1630:

1. Откройте `main.py`
2. Добавьте после импортов:
```python
import os
os.environ["CUDA_VISIBLE_DEVICES"] = "1"
```

GPU 0 = CMP 50HX, GPU 1 = GTX 1630 (обычно)

Проверьте порядок:
```python
import torch
print(torch.cuda.get_device_name(0))
print(torch.cuda.get_device_name(1))
```

## Автозапуск приложения

1. Создайте файл `start_shadowspeaker.bat`:
```batch

python main.py
pause
```

2. Создайте ярлык на рабочем столе
3. Для автозапуска поместите ярлык в:
   ```
   shell:startup
   ```
